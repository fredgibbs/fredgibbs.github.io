# Slide Images

The deck in `../index.html` expects three images in this folder. Drop in
whatever you find — the filenames below are what the `<img src=…>` tags
reference.

## Expected filenames

- `tycho-astronomia-instaurata-mechanica-1598.jpg`
  Frontispiece / plate from Tycho Brahe's *Astronomia instaurata mechanica* (1598).
  Good sources: Smithsonian / Dibner Library, Wikimedia Commons.
  Search: **"Tycho Brahe Astronomia instaurata mechanica frontispiece"**

- `galileo-sidereus-nuncius-corrected-pages.jpg`
  Corrected page from Galileo's *Sidereus Nuncius* (1610) showing the Medicean
  paste-over, or a moon engraving.
  Good sources: Museo Galileo (Florence) digital collections, Library of Congress.
  Search: **"Galileo Sidereus Nuncius corrected pages"** or
  **"Sidereus Nuncius Medicean stars paste"**

- `wicked-bible-1631-commandment-page.jpg`
  The Exodus 20 page from the 1631 "Wicked Bible" showing the missing "not".
  Good sources: British Library digital collections, Wikimedia Commons.
  Search: **"Wicked Bible 1631 commandment page"**

## Notes

- Landscape or portrait both work — the CSS caps `max-height: 72vh` so tall
  images won't overflow.
- JPG or PNG is fine; keep files under ~1 MB where possible.
- If you change a filename, update the matching `<img src="…">` in `../index.html`.
